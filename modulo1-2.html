Modulo 1: Fundamentos PHP

Sintaxis básica de PHP

PHP utiliza etiquetas de apertura y cierre para delimitar el código PHP en un archivo. Las etiquetas estándar son:

  • Etiqueta de apertura: <?php
  • Etiqueta de cierre: ?>

Todo el código PHP debe estar contenido dentro de estas etiquetas para que el servidor lo interprete correctamente.

Comentarios

En PHP, los comentarios se pueden utilizar para documentar el código y hacerlo más legible. Hay dos tipos de comentarios:

  • Comentarios de una línea: Se crean con // y todo lo que sigue en la línea se considera un comentario.
  • Comentarios de varias líneas: Se crean con /* */ y pueden abarcar múltiples líneas de código.

Variables y tipos de datos

Las variables en PHP se declaran con el símbolo $ seguido del nombre de la

variable. PHP es un lenguaje de tipado dinámico, lo que significa que no es necesario declarar el tipo de una variable antes de usarla. Algunos tipos de datos comunes en PHP son:

  • Cadenas de texto: Representan texto y se pueden definir con comillas simples (' ') o dobles (" ").

  • Números: Pueden ser enteros o de punto flotante (números con decimales).

  • Booleanos: Representan verdadero o falso.

  • Arrays: Estructuras que almacenan múltiples valores en una sola variable.

  • Objetos: Instancias de clases que contienen propiedades y métodos.
